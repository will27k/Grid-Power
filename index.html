GRID POWER

Game Rules

Click on white tiles to flip them and adjacent tiles.

Black tiles cannot be clicked.

Win by turning all tiles black!

Power Tiles:

Red tile: Flips diagonally in an X pattern (center + 4 diagonal corners)

Blue tile: Only flips adjacent tiles in a + pattern (not itself)

Green tile: Can be clicked even when black (+ pattern)

Purple tile: Flips all tiles in its row and column

Moves: 0
